# taehwan's note

알고리즘 문제 풀이와 개발 학습 과정을 기록하는 개인 블로그입니다.

**[taehwanis.github.io](https://taehwanis.github.io)**

## Stack

- **Framework**: Jekyll (Minimal Mistakes)
- **Theme**: Terminal (custom) — Tokyo Night palette
- **Hosting**: GitHub Pages
- **Comments**: giscus

## Running Locally

```bash
gem install bundler jekyll
bundle install
bundle exec jekyll serve
# http://localhost:4000
```
